# Platzhalter für 7-Zoll-Tablet-Screenshots (Deutsch)

Bitte platzieren Sie hier Screenshots mit folgenden Anforderungen:
- Mindestens 600px breit
- Empfohlene Auflösung: 1200x1920 Pixel
- Format: PNG oder JPEG
- Dateinamen: screenshot_1.png, screenshot_2.png, usw.

Benötigte Screenshots (Vorschläge):
1. Hauptbildschirm mit Verbindungsliste
2. Verbindungsdetails / Hinzufügen einer neuen Verbindung
3. Datei-Browser mit Dateien und Ordnern
4. Dateioperationen (Kontextmenü)
5. Einstellungen (falls vorhanden)

Hinweis: 
- Stellen Sie sicher, dass keine sensiblen Informationen oder echte Netzwerknamen in den Screenshots sichtbar sind.
- Für Tablets sollte das Layout die größere Bildschirmfläche optimal nutzen (z.B. durch Zwei-Spalten-Layout, falls implementiert).
